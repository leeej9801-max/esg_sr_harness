# Glossary

- Issue: ESG 평가와 보고의 상위 주제
- Metric: Issue를 측정하는 저장·추적 기준 단위
- KPI Fact: 회사·기간 기준 실제 값
- Evidence: 문서 기반 근거 원문
- Chunk: Evidence의 검색 단위
- Signal: 외부 징후 입력
- Judgment: 설문/인터뷰 기반 사람 판단
- Impact Score: 환경·사회 영향 관점 점수
- Financial Score: 재무 영향 관점 점수
- Sub-topic: 보고서 문단 주제 단위
- Prompt Template: 생성 입력 규약
- Decision Log: 중요한 설계 결정 기록
- Validation Report: 검증 결과 보고 문서
- Handoff Note: 다음 작업 전달 문서
